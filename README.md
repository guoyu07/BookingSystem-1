# BookingSystem
PHP and SQL based room booking system

Dependencies :- 
- JQuery >= 3.2.1
- JQuery-UI >= 1.11.14
- JQuery.DataTables >= 1.10.3
- OPGSLib >= 2
  - ADFS >= 2
  - Email >= 2
  - LDAP >= 6
  - Log >= 3
  - Settings >= 4
  - SQL >= 1
- PHP >= 7
  - Wincache Extension
  - MySQL drivers
- MySQL

JQuery dependencies should go in /js folder
